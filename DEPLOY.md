# Number Battle — Deploy to Vercel

## 1. Set up Upstash Redis (free)

1. Go to https://console.upstash.com and sign up (free)
2. Click **Create Database** → choose a region → create
3. On the database page, scroll to **REST API** section
4. Copy the **UPSTASH_REDIS_REST_URL** and **UPSTASH_REDIS_REST_TOKEN** values
